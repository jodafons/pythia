
# Pythia 8 Package (2.44)

Pythia event generator


## Installation

```bash
mkdir .bin && .bin
git clone https://github.com/jodafons/pythia.git && cd pythia
./configure
make -j4
export PYTHIA8_LIBRARY=${PWD}/lib/archive
export PYTHIA8_INCLUDE=${PWD}/include
```
